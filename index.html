Welcome to eXpansion

eXpansion is a Dedicated Server Controller for ManiaPlanet games by Ubisoft/NADEO, adding very cool features to the online gaming experience!

eXpansion V1 support was dropped during MP#4 Beta; it required a total overhaul before it could become compatible with the latests version of the game. eXpansion V1 contained quite a long list of technical dept coming from usage of unmaintained libraries as well as design errors.

At that point we decided that it was impossible for 2 people to work it to the new standarts while keeping all the functionalities. And therefore started the development of eXpansion V2.

eXpansion V2 is still in developmment and we don't expect a release even alpha anytime soon. We wish tto put in place solid bases before starting to create add functionnalities to the system. We don't expect the second version to ever have as much feature as the first version, we are going to rely on the community for that.

eXpansion 2 is Symfony based. This allows eXpansion 2 to be very modular.
The Manialink engine uses steeffeen's Fancy Manialink's library with some custom parts to inject maniascript.
