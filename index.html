Welcome to eXpansion

eXpansion is a Dedicated Server Controller for ManiaPlanet games by Ubisoft/NADEO, adding very cool features to the online gaming experience!

eXpansion V1 support was dropped during MP#4 Beta; it required a total overhaul before it could become compatible with the latests version of the game. eXpansion V1 contained quite a long list of technical dept coming from usage of unmaintained libraries as well as design errors.

At that point we decided that it was impossible for 2 people to work it to the new standarts while keeping all the functionalities. And therefore started the development of eXpansion V2.

eXpansion V2 is still in developmment and we don't expect a release even alpha anytime soon. We wish tto put in place solid bases before starting to create add functionnalities to the system. We don't expect the second version to ever have as much feature as the first version, we are going to rely on the community for that.

eXpansion 2 is Symfony based. This allows eXpansion 2 to be very modular.
The Manialink engine uses steeffeen's Fancy Manialink's library with some custom parts to inject maniascript.

Top features to expect :

Good code; Using Symfony principles

Basically it means the experience you will get in coding for eXpansion can serve you every where else and even on your CV. More

Anyone that has developped in Symfony before will have no trouble writing bundles(plugins) for eXpansion.
But it also means that what you are going to learnt to code for eXpansion can be used to code a website based on symfony.

Light & Fast

Maniascript allows to delagete tasks and take of the load from the sever. More

eXpansion2 send a single manialink to update all widgets, maniascript handles player related data.
It will also group manialinks in an intelligent fashion to create as little as possible load to the dedicated server.

International

eXpansion supports multilingual messages. So translating eXpansion for a new language isn't hacking. It just part of what it is meant to do. Contact us if you wish to work on the translations

Keep updated, easy

Thanks to Composer you can keep updated with the latest stable, nightly or subscribe even to the developer build!

Compatible

Trackmania or Shootmania, eXpansion is built to be extend to work on any environment & on any gamemode. More

Unique data provider based architecture coupled with Smyfony DI makes adapting existing plugins to new gamemodes easier then ever.
Exemple : If changes are made to make LocalRecords compatible with a new gamemode, the localRecordsWiget will automatically start working.

Easy to Install

eXpansion Add-on is easy to install, even for novice server administrator with our install tutorial!

Well documented

Very complete documentation with various exemples.

Unit tested

A united tested core, it means less bugs, less crashes.
